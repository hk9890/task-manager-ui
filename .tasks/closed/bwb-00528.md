## Description
Add a lightweight repository-managed pre-commit hook patterned after ai-config-manager's staged-Go formatting hook, adapted to this repo's needs.

## Instructions
1. Add a repo-local pre-commit hook script that discovers staged `.go` files, runs `gofmt -w` on them, and re-stages the formatted files.
2. Keep the hook intentionally fast and narrow; do not turn it into a full test runner.
3. Provide a straightforward Makefile-independent install path using `git config core.hooksPath scripts/git-hooks`.
4. Document how developers install and use the hook.

## Files to Modify
- `scripts/git-hooks/pre-commit`
- `docs/CHANGE-WORKFLOW.md` or another clearly discoverable workflow doc

## Explicit Non-Goal
- The hook should not depend on the Makefile to be usable.
