This produces the stuttered qualified name dashboard.DashboardDefinitionProvider at every call site (confirmed in internal/mode/board/model.go:40,59,356). Go naming convention (Effective Go, CodeReviewComments) says: avoid stuttering package name in exported identifiers. Should be dashboard.Provider or dashboard.DefinitionProvider.
