## Description
Verify the new runtime configuration system works end to end for config loading, launcher overrides, and fully configurable keybindings before the epic is closed.

## Verification Steps
1. Run BWB with no config file and confirm startup still uses defaults.
2. Run BWB with a representative config that overrides `editor.command`, one built-in launcher action, shell/board keys, and search/detail/modal keys.
3. Confirm unknown YAML keys generate warnings but do not prevent startup.
4. Confirm invalid YAML, invalid key names, and conflicting keybindings fail startup with clear errors.
5. Run the required quality gates.

## Acceptance Criteria
- [ ] All implementation tasks in this epic are closed.
- [ ] A representative config file successfully overrides editor, launcher, and keybinding settings.
- [ ] Unknown YAML keys generate startup warnings without aborting launch.
- [ ] Invalid config and conflicting keybindings fail startup with clear errors.
- [ ] Quality gates pass: `go build ./cmd/bwb`, `go vet ./...`, and `go test ./...`.
