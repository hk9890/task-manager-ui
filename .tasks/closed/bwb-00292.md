## Summary

`FakeBeadsGateway.ListIssues` and `FakeBeadsGateway.Query` ignore `SortBy`/`SortOrder` in the query/opts structs. The contract test `Invariants/ListIssues/SortApplied` accepts "either direction" to paper over this. This is a documented limitation of the fake (deterministic test double) but should be noted in the fake's doc comment so callers know sort-order tests must use the real gateway.
