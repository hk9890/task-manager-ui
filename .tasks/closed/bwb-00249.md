## Symptom

Records in `~/.local/state/bwb/bwb.log` from the dashboard component contain
the `component` key twice:

```
{"...","component":"dashboard","component":"dashboard","group":"Closed",...}
```

263 occurrences observed (all "backend sort assumption broken" warnings).

## Root Cause

The logger handed to the board mode already carries `component=dashboard`
(set in `cmd/bwb/main.go:46`: `services.Logger = opts.logManager.Component("dashboard")`).

`internal/mode/board/model.go:397` then redundantly adds it again:

```go
logger := m.logger.With("component", "dashboard")
```

slog's default JSON handler emits both keys.

## Fix

Drop the redundant `.With("component", "dashboard")` in
`internal/mode/board/model.go:397`, OR rename it to a sub-component label
(e.g. `"subcomponent", "compose"`) if the intent was scoping.

## Acceptance Criteria

- No JSON record in `bwb.log` contains a duplicated `component` key.
- Test (e.g. capturing the slog handler output) asserts single occurrence.
