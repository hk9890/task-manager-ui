The project is on Go 1.24.2 (go.mod). There are 3 separate private `maxInt(a, b int) int` implementations:
- internal/app/model.go:1314
- internal/mode/details/model.go:142
- internal/ui/modal/modal.go:412

Go 1.21 introduced the built-in `max()` function. All three copies should be replaced with the stdlib `max()` and the private functions deleted.

This is simple cleanup — fewer private helpers, fewer things to maintain.
