## Description

After this epic, every agent skill, slash-command, or guidance file that says `make X` or bare `go test ./...` is stale. Sweep and update.

Reviewer finding F30. Depends on i1th.6 (docs ticket establishes the new vocabulary first).

## Files to Modify

- `.claude/` (any subdirectory — slash commands, agent definitions, etc.)
- Any `bd remember` entries that mention test commands (`bd remember list` to enumerate)
- Repo-local AGENTS.md / CLAUDE.md / `docs/` (cross-check with i1th.6's work)

## Instructions

1. Enumerate references:
   ```bash
   grep -RnE '(make [a-z]+|go test ./\.\.\.)' .claude/ AGENTS.md CLAUDE.md docs/ 2>/dev/null
   bd remember list 2>&1 | grep -E 'make |go test' || echo "no bd remember hits"
   ```
2. For each hit, update to the mise equivalent. Common mappings:
   - `make build` → `mise run build`
   - `make test` → `mise run test:all` (or `mise run test` if unit-only is wanted — be deliberate)
   - `make quality` / `make all` → `mise run quality`
   - `go test ./...` (when meaning "run all tests") → `mise run test:all`
   - `go test ./...` (when meaning "run unit tests") → `mise run test`
3. Update `bd remember` entries via `bd remember add/edit` for any that reference legacy commands.
4. Re-run the grep at the end and confirm zero hits.

## Acceptance Criteria

- [ ] `grep -RnE '(make [a-z]+|go test ./\.\.\.)' .claude/ AGENTS.md CLAUDE.md docs/` returns zero hits
