## Description
Full codebase review identified substantial dead code across multiple packages. Removing it improves navigability and reduces maintenance burden without changing any behavior.

**NOTE:** This task is blocked by beads-workbench-xcp (overlay tests) because item 11 below removes overlay `BottomLeft`/`Top` position constants that xcp needs to test first. Only delete overlay position constants AFTER xcp confirms which are actually used by tests.

## Instructions
Remove the following dead code items:

### app package
1. `internal/app/model.go` — Delete `headerHelpLines()` function (discards args, returns nil)
2. `internal/app/model.go` — Delete `modeSwitchKey()` function (never called)
3. `internal/app/services.go` — Delete `SharedUIState` struct and all its allocations (allocated everywhere, read nowhere; toast state lives in `Model.toast`)

### config package
4. `internal/config/keybindings.go` — Delete `ModeBoardKey()`, `ModeSearchKey()`, `ModeDetailKey()`, `Label()` methods and the `keysByID` map (zero callers outside definition file)
5. `internal/config/load.go` — Inline or delete `errorsIsNotExist` wrapper (redundant since Go 1.16 — `os.IsNotExist` already calls `errors.Is`)
6. `internal/config/load.go` — Remove stale entries from `allowedSequenceMappingKeys` that reference unreachable keybinding nodes

### dashboard package
7. `internal/dashboard/builtin_provider.go` — Delete `actorResolver` field, `ActorResolver` interface, `EnvActorResolver` struct, and `NewBuiltInProviderWithActorResolver` constructor (stored but never read)
8. `internal/dashboard/builtin_provider_test.go` — Delete `TestBuiltInProviderDashboardsWithActorRemainsFourLaneDefault` (tests dead code, asserts same result as the non-actor test)

### ui packages
9. `internal/ui/styles/format.go` — Delete `FormatCommentIndicator()` (zero production callers)
10. `internal/ui/styles/format_test.go` — Delete the test for `FormatCommentIndicator`
11. `internal/ui/overlay/overlay.go` — Remove `BottomLeft` and `Top` **position constants** (iota values in the `Position` type) and their `case` branches in `calculatePosition`. These are iota constants, NOT standalone functions. Only remove them AFTER overlay tests (xcp) confirm they aren't needed for test coverage.
12. `internal/ui/styles/section.go` — Delete `TopLeftHint`, `BottomLeft`, `BottomRight` fields from `FormSectionConfig` and their rendering logic (~50 lines) — no caller sets these
13. `internal/ui/board/board.go` — Remove dead `availableWidth` variable (lines ~57-59, immediately shadowed)

### gateway package
14. `internal/gateway/beads/writes.go` — Remove `commandRunner()` nil-guard method (lines ~18-24, constructor prevents nil, read methods skip this guard — inconsistent)

### launcher/editor package
15. `internal/launcher/editor/service.go` — Remove `if e == nil` nil-receiver guard (line ~57-59, unreachable since constructor prevents nil)

After each removal, run `go build ./cmd/bwb` to verify compilation. Run `go test ./...` after all removals.
