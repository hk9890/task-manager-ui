## Description
Deliver the standalone editing and launcher workflows that make Beads Workbench a tool hub rather than just a viewer. This epic covers external editor handoff, create/update/close/comment actions, launcher commands, and test coverage for the integrated flows.

## Goals
- Add external editor issue editing
- Add create/update/close/comment actions from the app
- Add launcher framework and built-in launcher actions
- Reuse donor editor helpers only where they fit the new architecture
- Add editor and launcher integration tests

## Success Criteria
- [ ] External editor handoff works for rich issue editing
- [ ] Users can create, update, close, and comment on issues from the app
- [ ] Launcher commands can interpolate issue context
- [ ] Built-in launchers exist for common terminal workflows
- [ ] Editor/launcher tests pass
