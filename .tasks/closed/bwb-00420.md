## Description
Deliver the core browsing experience for Beads Workbench: built-in dashboards, search without BQL, and issue details. This epic should make the standalone app useful as a daily issue browser.

## Goals
- Add built-in dashboard definitions for common work queues
- Deliver board/dashboard rendering through gateway-backed queries
- Deliver search with text and structured filters, without BQL
- Deliver issue detail rendering with metadata, comments, and dependency context
- Add browsing integration tests

## Success Criteria
- [ ] Built-in dashboards cover ready, active, blocked, and recent work
- [ ] Search works without BQL and supports text plus structured filters
- [ ] Issue details show official-surface metadata and related context
- [ ] Board, search, and detail flows are integrated into the app shell
- [ ] Browsing tests pass
