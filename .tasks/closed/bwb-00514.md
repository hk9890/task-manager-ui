Update the Go toolchain pin from 1.24.2 to 1.26.3 (matches dtctl's pin and current upstream stable). bwb already manages Go via `mise`, so this is a one-line bump plus a verification pass.

- `.mise.toml` — change `go = "1.24.2"` to `go = "1.26.3"`
