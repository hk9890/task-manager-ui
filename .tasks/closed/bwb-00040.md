## Description
Create a single logging surface for Beads Workbench using Go's `log/slog`.

This work should replace scattered ad-hoc stderr/debug logging with a central `internal/logging` package while keeping stderr as the operator-facing surface for warnings and failures. Diagnostics should also be written to a durable machine-readable JSON Lines log file in the default XDG state location, with built-in rotation and per-session correlation.

## Decisions Already Made
- [x] Use Go `log/slog` for logging
- [x] Create `internal/logging` as the central package
- [x] Keep a standard persistent log file with JSON Lines output
- [x] Do not add `--log-file` or `--log-format` configurability in this phase
- [x] Human-facing warnings and errors are written to both `stderr` and the persistent JSON Lines log file
- [x] `--debug` enables DEBUG-level logging and mirrors debug diagnostics to `stderr`
- [x] The existing `[bwb-debug]` stderr prefix is preserved for compatibility
- [x] Log rotation uses `gopkg.in/natefinch/lumberjack.v2`
- [x] Fixed rotation defaults are: max size 10 MB, max backups 5, max age 30 days, compress enabled
- [x] If the XDG state directory or log file path is unavailable or unwritable, BWB emits a single `stderr` warning and continues with `stderr`-only logging
- [x] Session IDs use short random hex generated once per run, are included in every structured record, and are printed once to `stderr` during debug startup so operators can correlate runs
- [x] Gateway execution traces log full argv in local file logs

## Success Criteria
- [ ] `internal/logging` exists and is the single application logging entrypoint
- [ ] Each BWB run has a session identifier attached to diagnostic records
- [ ] Diagnostics are written to a durable JSON Lines log file under XDG state
- [ ] Log rotation is enabled with fixed built-in defaults
- [ ] Startup/CLI diagnostics and gateway command traces use the central logging package
- [ ] Operator-facing stderr behavior remains clear and intentionally defined
- [ ] Tests and required verification pass
