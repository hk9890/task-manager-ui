scenarios_controller.go (applyControllerCmd, lines 28-41) is a near-copy of scenarios.go (applyCmd, lines 69-82). The only difference is the receiver type: tea.Model vs mode.Controller. Both share the same drainCmd. This is a textbook case for a Go generic function:

func applyCommands[T interface{ Update(tea.Msg) (T, tea.Cmd) }](model T, cmd tea.Cmd) T

This eliminates the duplication and makes it impossible for the two to drift. If generics are not desired (Go 1.18+ is required), at minimum add a comment linking the two implementations to prevent them from diverging independently. The Controller variant has no test file (scenarios_controller_test.go does not exist), which increases the drift risk.
