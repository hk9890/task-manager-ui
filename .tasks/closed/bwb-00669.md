## Instructions
1. Confirm `git status` is clean and `main` is up to date with `origin/main`.
2. Verify `git tag --list "v*"` does not already contain `v0.3.0`.
3. Create an annotated tag: `git tag -a v0.3.0 -m "bwb v0.3.0"`.

5. Watch the GitHub `Release` workflow until it completes successfully.
