## Summary

`FakeBeadsGateway.CloseIssue` only sets `Status="closed"` but never sets `ClosedAt`. Real bd 1.0.4 always sets `closed_at` and it is visible via `ShowIssue`. UI tests using the fake to observe close metadata see zero `ClosedAt`.

## Discovered during

`beads-workbench-ibif` Write audit (parent epic: `beads-workbench-yspw`).

## Fix

In the fake's CloseIssue method:
```go
existing.ClosedAt = time.Now()
```
