## Description
Move startup and CLI-path diagnostics onto the central logging package without breaking the documented operator-facing CLI behavior.

This task should preserve current stdout/stderr contracts while ensuring startup resolution events, config warnings, and startup failures are emitted through the new logging path as structured diagnostics.

## Instructions
1. Initialize logging early in `cmd/bwb/main.go` so startup code has access to the central logger.
2. Replace ad-hoc debug emission in the startup path with structured logging events from `internal/logging`.
3. Implement the finalized stderr/debug behavior defined by the epic: warnings/errors go to both stderr and the JSON log, `--debug` enables DEBUG-level logging and mirrors debug diagnostics to stderr, and the existing `[bwb-debug]` prefix is preserved.
4. Keep the human-facing stderr behavior clear for warnings and failures, and preserve the documented non-interactive stdout/stderr contracts.
5. Make sure non-interactive flows (`--help`, `--version`, `--print-config`, `--check-config`) continue to honor their existing stdout/stderr and exit-code contracts.
6. Ensure non-interactive help/version-style paths do not create a persistent log file as a side effect.
7. Update the architecture guardrail coverage in `cmd/bwb/architecture_guardrails_test.go`, which currently hardcodes `[bwb-debug]` stderr lines, so the test matches the finalized logging contract.
8. Add or update tests in `cmd/bwb/*_test.go` to prove the CLI behavior still works while central logging is active.

## Acceptance Criteria
- [ ] Startup/CLI diagnostics flow through `internal/logging`
- [ ] Existing non-interactive stdout/stderr and exit-code contracts remain intact
- [ ] Debug/startup stderr behavior matches the finalized logging contract and is test-covered
- [ ] Non-interactive help/version-style paths do not create the persistent log file
- [ ] The architecture guardrail test passes under the new logging contract
- [ ] CLI tests cover the new logging integration

## Files to Modify
